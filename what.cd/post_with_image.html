it's very nice, but has a few problems. you should add the following so that lightbox works:

.curtain {
display: none;
position: fixed;
top: 0%;
left: 0%;
width: 100%;
height: 100%;
background-color: black;
z-index: 1001;
-moz-opacity: 0.8;
opacity: .80;
filter: alpha(opacity=80);
}

.lightbox {
display: none;
position: fixed;
text-align: center;
top: 5%;
left: 5%;
width: 90%;
height: 90%;
padding: 0px;
z-index: 1002;
overflow: auto;
}



you haven't assigned anything for ".show_torrents" and ".hide_torrents", so a large part of the torrent page functionality is missing.

Edit/ Anyone who uses this stylesheet will be able to submit nothing except lowercase text, which I think is a big problem.
